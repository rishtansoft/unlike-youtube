# YouTube Unlike Helper

YouTube'dagi yoqtirilgan videolarni avtomatik o'chiradigan Chrome extension.

## O'rnatish

1. Chrome brauzerida `chrome://extensions/` sahifasini oching
2. O'ng yuqori burchakda "Developer mode" ni yoqing
3. "Load unpacked" tugmasini bosing
4. Ushbu papkani tanlang

## Ishlatish

1. https://www.youtube.com/playlist?list=LL sahifasiga o'ting
2. Sahifa to'liq yuklanishini kuting
3. Extension avtomatik ravishda birinchi 5 ta videoni o'chiradi

## Eslatma

Extension faqat yoqtirilgan videolar sahifasida ishlaydi.
